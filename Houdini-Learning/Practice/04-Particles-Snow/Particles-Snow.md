
# Задача: Симуляция снега

> Уровень: 🟠 Средний  
> После изучения: [[04-Particles-and-DOP]]
> Цель: Создать базовую сцену с падающими частицами (снег/дождь).

---

## 🎯 Что нужно сделать

### Часть 1: Подготовка сцены (SOP)
1. Создайте **Grid** (20×20) → назовите `ground`.
2. Создайте **Box** над сеткой → назовите `emitter_area`.

### Часть 2: DOP Network
3. Создайте **DOP Network** (`Tab` → `dopnet`).
4. Внутри:
   - Добавьте **POP Source** → подключите `emitter_area` как источник.
     - Birth Type: **Impulse**
     - Impulse Activation: `1`
     - Number of Particles: `500`
   - Добавьте **POP Solver**.
   - Добавьте **Gravity**.
   - Добавьте **Static Object** → подключите `ground`.
5. Настройте **коллизии**: убедитесь, что частицы не проходят сквозь землю.

### Часть 3: Визуализация
6. Добавьте **DOP Import Records** в SOP → подключите выход DOP.
7. В Viewport:
   - Включите отображение частиц.
   - Настройте размер (`pscale`) через Attribute Wrangle, если нужно.
8. Сохраните как `solution.hipnc`.

---

## 💡 Подсказки
- Если частиц не видно — проверьте **Birth Frame**.
- Используйте **Cache Simulation** (в DOP Network → Cache Settings).
- Для снега: уменьшите скорость и добавьте **POP Drag**.
