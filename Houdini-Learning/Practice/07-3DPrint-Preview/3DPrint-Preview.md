
# Задача: Визуализация модели для 3D-печати

> Уровень: 🔴 Продвинутый / Прикладной  
> После изучения: [[CNC-Toolpath-Vis|Визуализация траектории CNC-инструмента]]
> Контекст: SOP + VEX + Geometry Analysis  
> Цель: Подготовить и проанализировать 3D-модель для печати: проверить overhangs, добавить поддержки, экспортировать в STL.
---

## 🎯 Что нужно сделать

### Часть 1: Импорт модели
1. Создайте простую модель (например, **Box с вырезом** или **Suzanne**).
   - Или импортируйте `.stl`/`.obj` через **File SOP**.
2. Убедитесь, что геометрия **замкнутая (watertight)** и **ориентирована правильно**.

### Часть 2: Анализ угла наклона (Overhang Detection)
3. Добавьте **Attribute Wrangle** (Run Over: **Primitives**).
4. Напишите VEX-код для расчёта угла между нормалью грани и вертикалью:
   ```c
   vector up = set(0, 1, 0);
   float angle = acos(dot(@N, up));
   float degrees = radiansToDegrees(angle);
   @overhang = degrees > 45 ? 1 : 0;
   ```
5. Добавьте Color → используйте атрибут `@overhang` для окраски:
    Красный (`1,0,0`) — проблемные зоны (>45°),
    Зелёный (`0,1,0`) — безопасные.

### Часть 3: Генерация поддержек (упрощённая)
6. Выделите проблемные примитивы:
   Используйте Group Expression: `@overhang == 1`.
7. Добавьте Ray → направьте лучи вниз (`Direction = 0,-1,0`) до стола (`Ground`).
8. Используйте **Lattice** или **PolyWire**, чтобы превратить лучи в «ножки» поддержек.
> 💡 Альтернатива: создайте **Volume Rasterize** → **VDB From Polygons** → **VDB Resample** → **VDB to Mesh** для плотных поддержек.

### Часть 4: Экспорт
9. Объедините модель и поддержки через Merge.
10. Добавьте File SOP → сохраните как print_preview.stl.

>💡 Советы
 - Используйте Grid как «стол печати» (Z=0).
 - Для точного анализа — увеличьте детализацию модели (Subdivide).
 - Проверьте масштаб: 1 единица = 1 мм (стандарт для 3D-печати).
 # Пример VEX для цветовой индикации угла: 
```c
vector up = {0,1,0};
float angle_deg = radiansToDegrees(acos(dot(normalize(@N), up)));
@Cd = fit(angle_deg, 0, 90, set(0,1,0), set(1,0,0)); // зелёный → красный
```
