
# Задача: Scatter и Copy — процедурное размножение

> Уровень: 🟡 Средний 
> После изучения: [[02-Nodes-and-Networks]]
> Контекст: SOP  
> Цель: Научиться равномерно распределять объекты по поверхности с помощью Scatter + Copy to Points.

---

## 🎯 Что нужно сделать

1. Создайте **Grid** (сетку) — это будет поверхность.
2. Добавьте **Scatter** → подключите к Grid.
   - Установите **Force Total Count = 50**.
3. Создайте **Box** — это будет объект для копирования.
4. Добавьте **Copy to Points**:
   - Подключите Box к **первому входу** (Geometry to Copy),
   - Подключите Scatter к **второму входу** (Points to Copy Onto).
5. Добавьте **Normal** перед Scatter → чтобы точки имели ориентацию.
6. Добавьте **Randomize Transform** после Box → чтобы кубы немного различались.
7. Организуйте сеть:
   - Группируйте логические блоки,
   - Добавьте Sticky Notes,
   - Завершите Null-нодой `OUT_scatter`.
8. Сохраните как `solution.hipnc`.

---

## 💡 Подсказки

- Если копии «лежат» плоско — не забудьте **Normal**!
- Используйте **Viewport Display → Point Display** для отладки точек.
