
# Задача: Основы VEX — первый код

> Уровень: 🟠 Начальный VEX  
> После изучения: [[05-VEX-Fundamentals]]
> Контекст: SOP → Attribute Wrangle  
> Цель: Написать 3 простых скрипта для управления геометрией через VEX.

## 🎯 Задания

### 1. Случайный цвет по точкам
- Создайте **Sphere**.
- Добавьте **Attribute Wrangle** (Run Over: **Points**).
- Напишите код:
  ```c
  @Cd = rand(@ptnum + 50);
  ```
### 2. Радиальное сжатие сетки
Создайте **Grid** (20×20 divisions).
Добавьте **Attribute Wrangle** (Points).
Напишите код:
```c
float d = length(set(@P.x, 0, @P.z));
```
```c
@P.y = fit(d, 0, 5, 1, 0.1);
```
### 3. Условное удаление геометрии
Создайте **Box**.
Добавьте **Attribute Wrangle** (Points).
Напишите код:
```c
if (@P.x > 0) {
    removepoint(0, @ptnum);
}
```

> 💡 Советы
  Используйте **Geometry Spreadsheet**, чтобы видеть атрибуты.
  Если геометрия исчезла — проверьте логику условия в `removepoint`.
  Экспериментируйте: замените `rand()` на `noise(@P)` или добавьте `@Time`.
