
# Задача: Процедурный куб

> Уровень: 🟢 Начальный  
> После изучения: [[01-Interface-Basics]]
> Контекст: SOP (Object / Geometry)  
> Цель: Освоить базовый workflow Houdini — создание, редактирование, подключение нод.

---

## 🎯 Что нужно сделать

1. **Создайте новую сцену** в Houdini (File → New → `SOP` context).
2. **Добавьте Box**:
   - Нажмите `Tab` в Network Editor.
   - Введите `box` → выберите **Box**.
3. **Настройте параметры** в Parameter Editor:
   - Size: `2x2x2`
   - Divisions: `4x4x4`
1. **Добавьте Subdivide**:
   - Снова `Tab` → введите `subdivide`.
   - Подключите выход Box к входу Subdivide.
5. **Добавьте Color** (чтобы видеть геометрию):
   - `Tab` → `color` → подключите после Subdivide.
   - В параметрах: Color = `(0.2, 0.6, 1.0)` (голубой).
6. **Навигация в Viewport**:
   - Обрамите объект: выделите любую ноду → нажмите `F`.
   - Покрутите камеру: `Alt + ПКМ`, переместите: `Alt + ЛКМ`.
7. **Сохраните файл** как: `Practice/01-Procedural-Box/solution_'you'.hipnc`
