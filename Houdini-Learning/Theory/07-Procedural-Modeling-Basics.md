## 🎯 Цель модуля

Понять **ключевые принципы процедурного подхода**:
- параметрическое построение геометрии,
- использование **Scatter**, **Copy to Points**, **Instance**,
- создание **вариативных** объектов через атрибуты,
- сборка повторно используемых **подсетей (Subnet / HDA)**.

---

## 🧠 Что такое процедурное моделирование?

**Процедурное** — значит описанное набором правил и параметров, а не нарисованное вручную.

| Традиционное моделирование | Процедурное моделирование |
|--------------------------|--------------------------|
| Изменил форму → переделывай всё | Изменил параметр → всё пересчиталось |
| Результат уникален и «заморожен» | Результат генерируется по правилам |
| Сложно тиражировать | Легко создавать вариации |

> 💡 В Equilibrium: процедурный подход = быстрая генерация деталей для CNC, тестовых геометрий для принтера, паттернов для лазера.

## 🔑 Три кита процедурного моделирования

### 1. Параметризация

Каждый параметр — ручка управления. Используйте **выражения** вместо чисел:

```
Size X:  ch("../controller/width")   ← ссылка на параметр другой ноды
Size Y:  $F * 0.1                    ← зависит от кадра анимации
```

> 💡 `ch()` — читает канал (параметр) любой ноды по пути.

---

### 2. Scatter + Copy to Points

Самый используемый паттерн в процедурном моделировании:

```
[Поверхность/Кривая]
       ↓
[Scatter]              ← генерирует случайные точки
       ↓
[Copy to Points] ← [Объект-шаблон (Box, Sphere, ...)]
       ↓
[OUT_result]
```

**Атрибуты, влияющие на Copy to Points:**

| Атрибут на точке | Что делает |
|-----------------|-----------|
| `@N` | Ориентирует копию по нормали |
| `@up` | Задаёт вектор «вверх» |
| `@scale` | Масштаб копии (float или vector) |
| `@orient` | Кватернион ориентации (точный контроль) |
| `@pscale` | Равномерный масштаб |
| `@id` | Выбор варианта из нескольких шаблонов |

---

### 3. Атрибуты как данные для управления

Атрибуты — это не только цвет и позиция. Это **данные для логики**:

```c
// Attribute Wrangle (Point) — случайный масштаб и поворот
@pscale = fit01(rand(@ptnum), 0.5, 2.0);    // масштаб от 0.5 до 2.0
@orient = quaternion(radians(rand(@ptnum + 99) * 360), {0,1,0}); // поворот Y
```

## 🧩 Ключевые ноды

| Нода | Назначение |
|------|-----------|
| **Scatter** | Случайные точки на поверхности |
| **Copy to Points** | Копирование геометрии на точки |
| **Instance** | Быстрое копирование (GPU, для больших сцен) |
| **For Each** | Цикл по примитивам или точкам |
| **Subnet** | Группировка нод в блок |
| **Null (с параметрами)** | «Пульт управления» для сети |
| **Point Replicate** | Быстрое дублирование точек |
| **L-System** | Генерация по правилам (фракталы, ветви) |

## 🛠️ Практика: параметрический забор

> Цель: забор из досок, управляемый тремя параметрами: длина, высота, шаг.

**Шаги:**

1. Создайте **Line** (направление X) с параметром `Length`.
2. Добавьте **Resample** → установите `Length` = шаг между досками.
3. Создайте отдельный **Box** (одна доска) — настройте пропорции.
4. Подключите через **Copy to Points** → доски встанут вдоль линии.
5. Добавьте **Attribute Wrangle**:
```c
// Небольшая случайная высота каждой доски
@scale.y = fit01(rand(@ptnum), 0.8, 1.1);
```
6. Создайте **Null** с именем `CTRL_fence` → добавьте spare-параметры: `length`, `step`, `height`.
7. Свяжите ноды через `ch()` с этими параметрами.

**Результат:** один ползунок меняет весь забор.

## 🔄 Подсети (Subnet) и HDA

Когда сеть разрослась — заверните в **Subnet**:
1. Выделите группу нод.
2. `ПКМ → Collapse into Subnet`.
3. Переименуйте → теперь это **переиспользуемый блок**.

**HDA (Houdini Digital Asset)** — следующий уровень:
- Subnet с опубликованными параметрами.
- Можно расшарить с партнёром (Александром).
- Появляется в меню `Tab` как обычная нода.

> 💡 В Equilibrium: создайте HDA «CNC-рамка» или «Паттерн для лазера» — и используйте в любом проекте.

## 📐 Процедурные паттерны для CNC/лазера

```c
// Attribute Wrangle (Point) — генерация сетки точек
int rows = 10;
int cols = 10;
float step = 0.5;

for(int r = 0; r < rows; r++) {
    for(int c = 0; c < cols; c++) {
        vector pos = set(c * step, 0, r * step);
        int pt = addpoint(0, pos);
        setpointattrib(0, "pscale", pt, fit01(rand(r*cols+c), 0.1, 0.4));
    }
}
```
