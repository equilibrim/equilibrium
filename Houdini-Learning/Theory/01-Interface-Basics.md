# Интерфейс Houdini: основы

> Актуально для SideFX Houdini **20.5–21.0** (2025–2026)  
> Уровень: 🟢 Начальный  
> Время на изучение: 1–2 часа

---

## 🎯 Цель модуля
Познакомиться с ключевыми элементами интерфейса Houdini, научиться:
- перемещаться в 3D-виде,
- создавать и редактировать ноды,
- понимать структуру сети,
- использовать контекстные меню и горячие клавиши.

---

## 🖥️ Основные панели

| Панель | Назначение | Горячая клавиша |
|-------|-----------|------------------|
| **Viewport (3D View)** | Основное окно просмотра сцены | `1` — режим перемещения, `2` — вращения, `3` — масштаба |
| **Network Editor** | Рабочая область для создания нод | `Tab` — вызов меню создания нод |
| **Parameter Editor** | Настройка параметров выбранной ноды | Автоматически появляется при выборе ноды |
| **Channel Editor / Dope Sheet** | Анимация и ключи | `F2` |
| **Render View** | Предпросмотр рендера | `F9` |

> 💡 Совет: нажмите `Space` в любом месте → появится **радиальное меню** с быстрым доступом к основным инструментам.

---

## 🔧 Основные горячие клавиши

| Действие | Клавиша |
|--------|--------|
| Создать ноду | `Tab` |
| Выделить всё | `A` |
| Отменить | `Ctrl+Z` |
| Повторить | `Ctrl+Y` |
| Сохранить | `Ctrl+S` |
| Переместить камеру (Viewport) | `Alt + ЛКМ` |
| Вращать камеру | `Alt + ПКМ` |
| Масштабировать | `Alt + СКМ` или колёсико |
| Обрамить выделенное | `F` |
| Обрамить всю сцену | `Home` |

> ⚠️ Houdini чувствителен к регистру! Например, `s` — scale (локально), `S` — scale (глобально).

---

## 🌐 Типы сетей (Contexts)

Houdini делит работу на **контексты** — разные типы сетей:

| Контекст | Префикс | Назначение |
|--------|--------|-----------|
| **SOP** | `/obj/geo1/` | Геометрия (моделирование) |
| **DOP** | `/dopnet1/` | Динамика (физика, частицы) |
| **VOP** | `/mat1/` | Шейдеры и материалы |
| **CHOP** | `/chopnet1/` | Аудио и анимационные каналы |
| **ROP** | `/out/` | Рендер и экспорт |

> ✅ Для начала сосредоточьтесь на **SOP** — это основа процедурного моделирования.

---

## 🛠️ Практические шаги (выполните в Houdini)

1. Запустите Houdini (Indie или Apprentice).
2. Создайте новый файл → **SOP context**.
3. Нажмите `Tab` → введите `box` → создайте **Box**.
4. Выделите Box → измените параметры в **Parameter Editor** (размер, деление).
5. Нажмите `Tab` → добавьте **Subdivide** → подключите к Box.
6. Используйте `F` → обрамьте объект в Viewport.
7. Сохраните файл как `practice-01-interface.hipnc` в папке `Practice/01-Procedural-Box/`.
