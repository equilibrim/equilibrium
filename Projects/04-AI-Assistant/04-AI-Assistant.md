## 📦 Компоненты

### 1. Continue + VS Code (✅ запущено)
- Ollama с моделями: `llama3.1:8b`, `qwen-coder`
- **Claude Sonnet** (облако) — для сложных запросов и анализа
- Groq 70B (облако) — резервный вариант

### 2. TG-бот (🔧 планируется)
- Realtime пуш идей прямо из телефона
- Протокол встреч / аудит сессий
- Еженедельный дайджест прогресса по проектам
- Будущее: голос → текст, работа с фото/видео для дашборда
- → [[TG-bot]]

### 3. Локальный ассистент (🔧 планируется)
- Запуск нужного софта под текущий проект (Houdini, VS Code, Obsidian)
- Запуск скриптов прямо из команды
- Виртуализация окружений
- → [[Local-assistant]]
