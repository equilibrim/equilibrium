# 🧠 Equilibrium

Единая система знаний и разработки для **Владислава** и **Александра**.

> «Equilibrium — наш интеллектуальный экзоскелет, который растёт вместе с нами.»

---

## � О проекте

Equilibrium — это персональная база знаний и инженерная экосистема, объединяющая обучение, проектирование и автоматизацию в одном репозитории. Система построена на Obsidian + Git и охватывает:

- обучение процедурному моделированию в SideFX Houdini
- подготовку к работе с CNC и 3D-печатью
- разработку скриптов автоматизации на Python
- построение локальной AI-инфраструктуры на базе VS Code + Continue + Claude

## 🚀 Быстрый старт

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/VladislavDesa/equilibrium.git
   ```
2. Открой папку `equilibrium` как Vault в Obsidian
3. Установи плагины из таблицы выше
4. Открой `Home.md` — это точка входа в систему
5. Запусти Ollama: `ollama serve`
6. Открой папку в VS Code для работы с Continue

## 🏷️ Соглашения

### YAML-метаданные проекта

Каждый проект в `Projects/` имеет главный файл с YAML в начале:

```yaml
---
status: in-progress
priority: 02-high
team: Vlad
---
```

### Статусы

| Значение | Описание |
|----------|----------|
| `planning` | Проектирование |
| `in-progress` | Активная разработка |
| `review` | Проверка |
| `on-hold` | Приостановлено |
| `done` | Завершено |

### Приоритеты

| Значение | Уровень |
|----------|---------|
| `01-critical` | Критичный |
| `02-high` | Высокий |
| `03-medium` | Средний |
| `04-low` | Низкий |

### Ответственные

| Значение | Участник |
|----------|----------|
| `Vlad` | Владислав |
| `Alex` | Александр |
| `All` | Оба |

### Задачи

Все задачи помечаются тегом `#task` в любом файле репозитория. Завершить задачу — `Ctrl+Enter` на строке в Obsidian. Глобальный дашборд всех задач — `Tasks.md`.
